# musicPlayer
Descripción
Este repositorio contiene el código fuente de un Music Player desarrollado en colaboración con mis compañeros utilizando React.js. El Music Player es una aplicación web que permite a los usuarios reproducir y gestionar su colección de música de forma sencilla y visualmente atractiva.

Tecnologías utilizadas
El Music Player fue desarrollado utilizando las siguientes tecnologías:

React.js: Una biblioteca de JavaScript utilizada para construir interfaces de usuario interactivas y reutilizables.
HTML: Lenguaje de marcado utilizado para estructurar y presentar los componentes de la aplicación.
CSS: Hojas de estilo que añaden personalización y estilización visual al Music Player.
Funcionalidades
El Music Player con React.js incluye las siguientes funcionalidades principales:

Reproducción de música: Los usuarios pueden reproducir la colección de música con controles de reproducción como play, pause, adelantar y retroceder.
Visualización de metadatos: Se muestra información relevante sobre las canciones, como el título, artista, duración y letra de las canciones, para una mejor experiencia de escucha.
Interfaz intuitiva: El Music Player presenta una interfaz de usuario intuitiva y atractiva, que permite a los usuarios navegar y encontrar fácilmente las canciones que desean reproducir.
Responsive Design: El diseño del Music Player se adapta a diferentes tamaños de pantalla, permitiendo una experiencia de usuario óptima en dispositivos móviles y de escritorio.
